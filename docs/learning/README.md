## 笔记结构

每个阶段的笔记尽量包含：

- 本阶段要解决的问题。
- 面向初学者的核心解释。
- 必须区分的相近概念。
- 常见误解与精确说法。
- 一份能够独立复述的阶段结论。
- 自测问题和下一阶段衔接。

只有完成复述或小练习后，阶段才会标记为完成。
